# hthblog-pages

## 项目简介

这是一个用于归档和发布多个独立 HTML 页面的仓库。仓库内包含若干静态页面文件，主要用于内容展示、信息整理或技术演示。本仓库不针对特定域名或网站，仅作为页面资源的集中存放与版本管理用途。

## 目录说明

```
hthblog-pages/
├── index.html          # 根目录入口页面（可选）
├── pages/              # 存放主要 HTML 页面文件
│   ├── example1.html
│   ├── example2.html
│   └── ...
└── assets/             # 页面所需的静态资源（图片、样式、脚本等）
    ├── css/
    ├── js/
    └── images/
```

- `pages/`：存放各独立 HTML 页面，可按功能或主题命名。
- `assets/`：存放页面引用的通用资源文件，便于统一管理。

## 页面归档说明

- 每个 HTML 页面均为独立文件，可单独打开运行，不依赖后端服务。
- 页面内容以静态展示为主，部分可能包含简单的前端交互功能。
- 归档的页面可能来源于项目演示、技术笔记、个人作品或学习练习，不保证所有页面持续更新。

## 维护说明

- 本仓库按需更新，不定期添加或修改页面内容。
- 欢迎提交 Issue 或 Pull Request 进行建议或改进。
- 请勿在页面中嵌入恶意代码、跟踪脚本或违反平台政策的内容。
- 仓库维护者保留对页面内容进行整理、删除或归档的权利。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `pages/` 目录下的 HTML 文件即可查看。
3. 如需部署到静态托管服务，可将整个仓库内容上传，并设置首页为 `index.html` 或指定页面。
